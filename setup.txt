Taller 1

1. Clonar o copiar el proyecto en:
C:\xampp\htdocs\Taller1

2. Entrar a la carpeta del proyecto desde la terminal:
cd C:\xampp\htdocs\Taller1

3. Instalar dependencias: 
composer install

4. Copiar el archivo .env.example y renombrarlo a .env

5. Configurar la base de datos en el archivo .env:

DB_DATABASE=nombre_base_datos
DB_USERNAME=root
DB_PASSWORD=

6. Generar la clave de la aplicación:

php artisan key:generate

7. Crear la base de datos en phpMyAdmin.

8. Ejecutar migraciones:

php artisan migrate

9. Iniciar el servidor:

php artisan serve

10. Abrir en el navegador:

http://127.0.0.1:8000/

La ruta principal redirige automáticamente al módulo de órdenes.

Rutas principales

/ → Vista inicial del módulo
/orders/create → Crear orden
/orders/list → Listado de órdenes

Notas:
Es obligatorio ejecutar las migraciones antes de usar el sistema.
No se incluyen datos precargados.
